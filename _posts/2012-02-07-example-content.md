title: Updating Bijoy Linux.
---


## Via wget

```shell
bash -c "$(wget https://raw.githubusercontent.com/bijoylinux/bijoy/master/update.sh -O -)"
```
## Via curl

```shell
bash -c "$(curl -fsSL https://raw.githubusercontent.com/bijoylinux/bijoy/master/update.sh)"
```
